# 驾考练习助手 (Driving Test Practice Skill)

> 在支持 Agent Skill 的 Agent 中直接使用，随时练习中国驾照科目一 & 科目四，内置 **13,555 道真题**，覆盖全部 **11 种车型**。

---

## 为什么选择这个 Skill？

| 对比项 | 驾考宝典等 App | 本 Skill |
| ------ | -------------- | -------- |
| 使用场景 | 需要切换到手机 App | 在写代码的 IDE 里直接练 |
| 交互方式 | 固定 UI 点击 | 自然语言对话，随叫随练 |
| 学习辅助 | 查看解析 | 解析 + 记忆口诀 + 个性化易错分析 |
| 费用 | 部分功能收费 | 完全免费 |
| 碎片时间 | 需专门打开 | 等编译、等部署的间隙就能刷几题 |

## 练习模式

### 基础模式

| 模式 | 说明 | 示例指令 |
| ---- | ---- | -------- |
| 顺序练习 | 按题号从头到尾刷，支持断点续练 | "顺序练习科目一" |

| 模式 | 说明 | 示例指令 |

## 附加功能

**记忆口诀** — 覆盖 15 个高频专题的经典口诀，做题时自动推送相关口诀：

> "假一吊二撤三醉五逃终生" — 驾照处罚年限速记
>
> "261 / 145 / 520" — 罚款金额速记
>
> "口五站三" — 停车距离速记

**易错率追踪** — 记录每道题的做题次数和正确率，精准定位薄弱环节。

**进度统计** — 总做题数、正确率、各分类掌握情况、模拟考试历史成绩，一句"看看我的进度"即可查看。

## 安装

本 Skill 采用标准 Agent Skill 格式，可在**支持 Agent Skill 的 Agent** 中直接使用，安装后 Agent 会自动加载并在对话中响应驾考相关请求。

### 环境要求

- Python 3.6+（仅使用标准库，无需 pip install）
- 支持 Agent Skill 的环境（如 Cursor、Claude Code 等）

### Cursor

```bash
# 方式一：作为个人 Skill（所有项目可用）
git clone https://github.com/your-repo/driving-test.git ~/.cursor/skills/driving-test

# 方式二：作为项目 Skill（仅当前项目可用）
cd your-project
git clone https://github.com/your-repo/driving-test.git .cursor/skills/driving-test
```

安装后重启 Cursor，在 Agent 对话中直接说「练习驾考」即可触发。

### Claude Code

```bash
git clone https://github.com/your-repo/driving-test.git ~/.claude/skills/driving-test
```

### 手动安装

1. 下载本仓库 ZIP 并解压
2. 将 `driving-test` 文件夹放到以下任一位置：
   - Cursor 个人级：`~/.cursor/skills/driving-test/`
   - Cursor 项目级：`<项目>/.cursor/skills/driving-test/`
   - Claude Code：`~/.claude/skills/driving-test/`
3. 确认 `SKILL.md` 位于 `driving-test/` 根目录下

## 题库更新

题库数据来自 jisuapi.com，如需更新到最新题目：

1. 浏览器打开 https://www.jisuapi.com/debug/driverexam/ 并登录
2. 从浏览器开发者工具中复制 Cookie，保存到 `scripts/cookies.txt`
3. 运行导入脚本：

```bash
python scripts/import_questions.py --type C1 --type A1 --type A2 --type D
```

4. 校验数据完整性：

```bash
python scripts/validate_questions.py
```

脚本会自动拉取所选车型的科目一和科目四全部题目，按分页逐批获取，去重后写入 `data/` 目录。

## FAQ

**Q：题目和真实考试一样吗？**
A：题库来源与驾考宝典等主流 App 相同，均来自公安部公开题库。但考试题目可能随政策更新，建议定期更新题库。

**Q：支持哪些车型？**
A：覆盖全部 11 种准驾车型：C1/C2/C3（小车）、A1/A3/B1（客车）、A2/B2（货车）、D/E/F（摩托车）。默认为 C1。

**Q：进度数据存在哪里？**
A：存在项目根目录的 `progress.json` 中，该文件已加入 `.gitignore`，不会被提交到版本控制。

**Q：可以在没有网络的情况下使用吗？**
A：可以。题库数据已离线存储在 `data/` 目录中，做题不需要网络。只有更新题库时才需要网络。

**Q：如何重置进度？**
A：删除项目根目录下的 `progress.json` 文件即可重新开始。
